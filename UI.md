# UIデザイン仕様書 - Brain Dump Partner

## 1. デザインコンセプト
**"Refresh & Focus" (リフレッシュと集中)**
ユーザーが頭の中を空っぽにし（Brain Dump）、重要なことに集中できるようなデザインを目指しました。

-   **テーマ**: 「朝の緑 (Morning Green)」
    -   スカイブルー、グリーン、イエローのグラデーションを使用し、爽やかな朝の始まりを表現しています。
-   **スタイル**: モダンでクリーンな「Nani.now」風のデザイン。
    -   柔らかい影（Soft Shadows）
    -   大きな角丸（Large Border Radius）
    -   浮遊感のあるカードレイアウト（Floating Cards）
-   **ダークモード**:
    -   夜間や暗い場所でも目に優しい「Deep Slate（深みのある青黒）」ベースの配色。

## 2. 主要コンポーネント

### 2.1 サイドバーナビゲーション
-   画面端から独立した「浮いている」カードデザイン。
-   ロゴ、ナビゲーションリンク、ユーザープロフィールを集約。

### 2.2 ダッシュボード
-   **エージェントステータス**: AIパートナーからの挨拶やアドバイスを表示。動的なグラデーション背景で存在感を演出。
-   **Top 3 Tasks**: 「今日やるべきこと」を3つに絞って表示し、優先順位を明確化。
-   **Weekly Progress**: タスク消化状況を可視化するシンプルなグラフ。

### 2.3 グローバルチャットウィジェット
-   画面右下に常駐するフローティングボタン（FAB）。
-   いつでも思いついたことを「書き出す（Dump）」したり、AIに相談が可能。

### 2.4 テーマ切り替え
-   画面右上のアイコンで「ライトモード（朝）」と「ダークモード（夜）」を切り替え可能。
-   設定はブラウザに保存されます。

## 3. 技術実装

### 3.1 技術スタック
-   **HTML5**: セマンティックなマークアップ。
-   **CSS3**: Vanilla CSS。**CSS変数**を全面的に使用し、テーマ切り替えを容易に実装。
-   **JavaScript**: 外部ライブラリなしのVanilla JS。

### 3.2 ファイル構成
-   `index.html`: 画面構造。
-   `styles.css`: スタイル定義。
    -   `:root`: ライトモードの変数を定義。
    -   `[data-theme="dark"]`: ダークモード時の変数上書き定義。
-   `script.js`: UIロジック（タブ切り替え、チャット開閉、テーマ切り替え）。

### 3.3 カスタマイズ方法
色は `styles.css` の `:root` ブロックで管理されています。ここを変更するだけで、アプリ全体のテーマカラーを調整できます。

```css
:root {
    --primary: #0ea5e9; /* メインカラー */
    --bg-main: #f0f4f8; /* 背景色 */
    /* ... */
}
```
